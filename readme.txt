High-fidelity Elementor button based on Uiverse (gharsh11032000) with default gradient and full Classic/Gradient compatibility.

== Description ==

This plugin adds a "Fancy Button" widget to Elementor, inspired by modern Uiverse designs. It features smooth transitions, an animated icon bubble, and full integration with Elementor's background controls (Classic and Gradient).

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Find the "Fancy Button" widget in the Elementor editor under the 'General' category.
