High-fidelity collection of premium buttons for Elementor. Includes Project Button, Uiverse Styles, and more.

== Description ==

This plugin adds a premium collection of button widgets to Elementor. Featuring the EWEB Project Button with smooth 0.3s diagonal animations and the EWEB Fancy Button inspired by Uiverse designs.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Find the EWEB widgets in the Elementor editor under the 'General' category.
