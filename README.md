## 🛠 Installation

1. Clone or download this repository.
2. Upload the `fancy-elementor-button-uiverse` folder to `/wp-content/plugins/`.
3. Activate the plugin in the WordPress admin panel.
4. Add the **Fancy Button** widget to any Elementor page.
